# SquadFind
Web App that allows gamers to find others with similar interests to play games with one another.
With so many different platforms and games, it can be difficult to find other players who enjoy all the games you do,
play complimentary to your play style, and are on at the same time as you. New players may also struggle to find more
experienced players who are willing to squad up with them, help them progress, and overall improve their skills. SquadFind
aims to fix this by allowing users to search for players based on certain criteria, or be recommended others based on
their inputted information.
## What users can do:
- Enter desired username
- Enter platform that this username is used for
- Select games that they are interested in playing
- Let others know what role they play in this game (Support, DPS, Tank, etc)
- Overall playstyle for each game interested in
- Experience level
- What times you typically play at
- Send friend requests and accept requests from other players
## Tech Stack:
- Java
- SpringBoot
- Maven
- PostgreSQL
- React
